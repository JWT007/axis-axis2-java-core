Axis 2.0 Release Notes

This is the First Milestone release of the Apache Axis2.

What's in this release?

This release includes the following features:

  • AXIOM, a SOAP specific streaming XML infoset model for SOAP 1.1/1.2 Messages
  • WSDL Object Model built based on the proposed WSDL 2.0 Component Model.
  • Handler framework with the support to specify the location of the Handler in a specific Phase
  • Support to bundle and enable the Handlers as Modules
  • Support for the Synchronous and Asynchronous IN-OUT messaging over HTTP transport
  • Hot deployment of Web Services based on the archived modules

What's still to do?

Please see a list of what we think needs doing - and please consider helping out if you're interested & able!

  • Encoding/data binding
  • JAX-RPC 1.1/2.0 compliance
  • WSDL Based code (server/client) code generator
  • MTOM attachments support
  • Support for other WSDL MEPs, both synchronously and asynchronously
  • Complete XML infoset support for AXIOM
  • Implementation of other transports. e.g. SMTP, TCP..
  • Handler framework with the support to specific the location of the a Handler relative to Phase
  • Extend the modules to act as plug-ins

Axis2 is taking shape, this M1 release is a glimpse what is to be expected in the future. If you happen to have vision of how the next generation of the Web Service should be and like to contribute to Apache Axis2 please help us to shape the Apache Axis2. Any contribution in the form of Coding, testing, submiting improvements to the documentation, and reporting bugs are always welcome.

Thanks for your interest in Axis2!

-- The Axis Development Team
